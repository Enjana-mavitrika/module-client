# MODULE CLIENT 
Auteur : _SOLOFO RABONARIJAONA_

## DESCRIPTIONS CONTENUS
* ``DOCUMENTATION/``
  * ``spécification des exigences.pdf`` : Document détaillant les spécifications des exigences liés au Module.
  * ``conception générale.pdf`` : Document détaillant l'analyse et conception du Module
  * ``planification.pdf`` : Estimation de l'implémentation du module
  * ``diagrammes/`` : Images des diagrammes de conception
* ``Rapport Analyse et Conception.pdf`` : Résumé du travail d'analyse et conception
* ``Rapport Implémentation.pdf`` : Résumé du travail d'implémentation
* ``SOURCE/``
  * ``app/`` : Application dynamique (FRONTEND & BACKEND)
  * ``prototype/`` : Prototype du module (FRONTEND)  

## Prérequis
* Microsoft Visual Studio 2019
* Navigateur web (Chrome, Firefox, Edge, etc, ...)

## Téléchargement
* Télécharger le fichier ZIP avec le bouton __Clone or download__ 
* Extraire l'archive
* Accéder au répertoire extrait : ``module-client-master``

## Lancement du prototype (statique FRONT)
* Accéder au répertoire ``/SOURCE/prototype/``
* Ouvrir le fichier ``index.html`` dans un navigateur web.

## Lancement de l'application (dynamique FRONT-BACK)
* Accéder au répertoire ``/SOURCE/app/``
* Ouvir la solution ``ModuleClient.sln`` avec Visual Studio 2019
* Démarrer le débogage de la solution ( ou appuyez sur la touche <kbd>F5</kbd> )
